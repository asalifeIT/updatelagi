TECHNOLOGY FEATURE

Angular JS

Asto Service Application (ASA) dibuat dengan menggunakan Framework Angular. Framework ini telah dipercaya oleh kalangan developer dalam membangun aplikasi yang kompleks dan andal serta dapat menjadi salah satu pilihan terbaik dalam membangun aplikasi cross-platform mobile dengan Ionic Framework.

Java Spring Boot

Asto Service Application (ASA) didukung dengan bahasa pemprograman Java dengan menggunakan framework Spring Boot pada sisi backend. Penggunaan framework Spring Boot dapat membantu dalam pembuatan aplikasi kompleks dengan security yang lebih baik dan aman.

DBMS PostgreSQL

Asto Service Application (ASA) menggunakan DBMS PostgreSQL dalam menyimpan data. Sistem basis data relasional ini bersifat open source yang telah dikembangkan selama 30 tahun serta telah mendapatkan reputasi yang baik dalam hal keandalan, ketahanan fitur, dan kinerja.

Ionic Framework

Asto Service Application (ASA) dibangun dengan Ionic Framework yakni sebuah framework aplikasi mobile berbasis HTML5 yang dapat digunakan untuk mengembangkan aplikasi mobile dengan teknologi web seperti HTML, CSS, dan Javascript.
